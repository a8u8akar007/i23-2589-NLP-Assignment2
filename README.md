# NLP Assignment 2

## Project Description
This project implements various NLP techniques from scratch, including TF-IDF, PPMI, Word2Vec, and sequence models like BiLSTM and a custom Transformer, for Assignment 2.

## Folder Structure
- `data/`: Contains raw and processed text data.
- `embeddings/`: Implementation of vector space models (TF-IDF, PPMI, Word2Vec).
- `models/`: Neural network architectures (BiLSTM, Custom Transformer).
- `utils/`: Helper functions for evaluation and visualization.
- `i23-2589_Assignment2_DS-X.ipynb`: Main experimentation and results notebook.

## How to Run
1. Ensure you have the required dependencies (PyTorch, NumPy, Matplotlib).
2. Open the `i23-2589_Assignment2_DS-X.ipynb` notebook in Jupyter or VS Code.
3. Run the cells sequentially to preprocess data, train models, and evaluate results.
